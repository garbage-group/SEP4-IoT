This header file provides the interface for interfacing with a DHT11 humidity and temperature sensor.

Fetches humidity and temperature readings from the DHT11 sensor. 

This function fetches the humidity and temperature readings from the DHT11 sensor. The function can only be called once every 2 seconds to avoid sensor lock-\/up.

To only get specific data, pass NULL for the arguments that are not needed. For example, to only get the humidity integer part, pass NULL for all other arguments.
